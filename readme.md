# WP react django app

This is a React - Django application for replacement of WP site.

## Installation

Use the package manager [pip](https://pip.pypa.io/en/stable/) to install foobar.

```bash
pip3 install -r requirements.txt
npm install
```

## Usage

```python

python3 manage.py runserver

```
